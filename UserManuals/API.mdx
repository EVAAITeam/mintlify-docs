## API Documentation

> ### Authentication

All API requests require authentication using your API key:

```
Authorization: Bearer YOUR_API_KEY
```

> ### Base URL

All API endpoints are available at:

```
https://api.eva-ai.cloud/v1
```

> ### Rate Limits

API rate limits are based on your subscription tier:

- Free: 100 requests/hour
- Pro: 1,000 requests/hour
- Enterprise: Custom limits

## Available Endpoints

> ### Contract Analysis

```
GET /contracts/{address}/analysis
```

Get comprehensive security analysis of a smart contract

> ### Token Information

```
GET /tokens/{address}/info
```

Retrieve detailed token information and metrics

> ### Market Data

```
GET /market/sentiment
```

Get market sentiment and trending data
