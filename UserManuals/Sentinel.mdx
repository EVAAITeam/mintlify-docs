## What is Sentinel?
Sentinel is EVA's browser extension designed to protect users from malicious websites, phishing attacks, and dangerous smart contracts in the DeFi space. It provides real-time security analysis and warnings to keep your crypto assets safe while browsing and interacting with decentralized applications.

## How Sentinel Works
> ### Automatic Scanning

Sentinel automatically scans every website you visit for potential security threats. It analyzes the domain, checks against known malicious sites, and evaluates any smart contracts present.

- Real-time domain reputation checking
- SSL certificate validation
- Malware and phishing detection

> ### Smart Contract Analysis

When you interact with DeFi protocols, Sentinel analyzes the smart contracts to identify potential risks:

- Contract verification status
- Honeypot detection
- Unusual permission requests
- High-risk function calls

> ### Risk Assessment System

## Using Sentinel

> ### Daily Browsing

Sentinel works silently in the background, providing protection without interrupting your browsing experience:

- Green shield icon indicates safe websites
- Yellow warning for potentially risky sites
- Red alert blocks dangerous websites
- Click the icon for detailed security reports

> ### DeFi Integration

When connecting to DeFi platforms, Sentinel provides enhanced protection:

- Pre-transaction security checks
- Token contract verification
- Liquidity pool analysis
- Gas fee optimization suggestions

## Troubleshooting & Support
> ### Common Issues

- **Extension not working**: Check if it's enabled in Chrome extensions and refresh the page
- **False positives**: Report the issue through the extension's feedback system
- **Performance issues**: Adjust security level or disable on trusted sites
