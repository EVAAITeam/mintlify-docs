<h3 style={{ color:"#ffffff" }}>
Overview

</h3>

EVA Sentinel is a powerful browser extension that provides real-time security scanning and protection for DeFi users. It automatically analyzes smart contracts, websites, and transactions to identify potential risks before you interact with them.

<h3 style={{ color:"#ffffff" }}>
Key Features

</h3>

- **Real-Time Contract Analysis:** Automatically scans smart contracts for vulnerabilities and malicious code
- **Phishing Protection:** Identifies and blocks malicious websites and fake DeFi platforms
- **Transaction Risk Assessment:** Evaluates transaction risks before execution
- **Community Alerts:** Receives warnings from the EVA community about known threats
